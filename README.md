# fzf_thumb


dependencies:

    - fzf, ( https://github.com/junegunn/fzf )
    - less
    - imagemagick display ( check repo , or build from source )

usage:

    - clone rep 
    - cp -additional- into .bashrc
    - put path/to/fzf_thumb into FZF_CTRL_T_OPS
    - enter your favourite shortcut into FZF_CTRL_T_OPS
    - profit
